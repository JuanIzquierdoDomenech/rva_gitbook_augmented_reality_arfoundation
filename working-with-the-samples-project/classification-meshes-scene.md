# Classification Meshes scene

This scene uses the [`ARMeshManager`](https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@6.2/manual/features/meshing.html) to generate and show the meshes generated from the environment, and the `MeshClassificationFracking` component to classify and apply a different material per mesh type:

* **Ceilings** are <mark style="background-color:blue;">blue</mark>
* **Doors** are <mark style="background-color:yellow;">yellow</mark>
* **Windows** are <mark style="background-color:purple;">pink</mark>
* **Unknown** things are <mark style="background-color:red;">red</mark>
